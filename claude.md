# Claude Code Instructions

This is a Jekyll-based personal blog hosted on GitHub Pages.

## Blog Posts

Posts are located in `_posts/` with the naming convention `YYYY-MM-DD-title.md`.

When writing or editing blog posts, refer to `AGENTS.md` for voice and style guidelines.

## Common Tasks

- **New post**: Create in `_posts/` with frontmatter (layout, title, date, categories)
- **Local preview**: `bundle exec jekyll serve`
- **Deploy**: Push to main branch (GitHub Pages auto-deploys)
